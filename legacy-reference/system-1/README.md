# النظام الأول القديم

هذا المجلد يحتوي على ملفات النظام الأول القديمة التي تم رفعها إلى المستودع كمرجع تاريخي وتقني.

المحتوى يشمل:
- `form/` نماذج Oracle Forms القديمة
- `rep/` ملفات التقارير القديمة
- `file/` ملفات مساندة
- `copy/` ملفات نسخ قديمة مرتبطة بالمصدر
- ملفات صور وأصول قديمة مرتبطة بالنظام

الغرض من هذا المجلد:
- الرجوع إلى الشاشات الأصلية عند بناء البديل الحديث
- مقارنة السلوك بين النظام القديم والجديد
- حفظ المصدر القديم في قسم منفصل وواضح داخل المستودع

تنبيه:
- هذا المجلد مرجعي فقط
- لا يمثل الكود التشغيلي الحالي
- النظام الجديد المعتمد موجود في `angular-project/` و`nestjs-backend/`
