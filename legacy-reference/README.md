# المرجع القديم

هذا المجلد مخصص لحفظ ملفات النظام القديم داخل المستودع بشكل منفصل عن النظام الجديد.

المجلدات المرجعية الموجودة هنا:
- `system-1/` النظام الأول القديم كما تم حفظه من المصدر
- `datg.rar` الأرشيف المضغوط المرجعي المعتمد للنظام الأول

مهم:
- هذا القسم مرجعي فقط
- لا يتم تشغيل النظام الجديد منه
- لا يتم تنفيذ التطوير الجديد داخله

المكان المعتمد للتطوير الحديث:
- `angular-project/`
- `nestjs-backend/`
